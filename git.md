git init - Inicialização de repositório
git add . - Adicionar todos os ficheiros ao repositório
git commit -m \<mensagem> - fazer commit repositório local
git remote add origin \<url> - adicionar o repositório remoto
git push origin \<master ou outro branch> - publicar código no repositório remoto (github)
git pull origin \<master ou outro branch> - publicar código no repositório local
git clone \<url> - clona o projeto localmente

Para utilizar um branch diferente do master

git checkout \<branch>
